# Meal-Tracker

This Angular.js application displays food intake information. Users may add, edit, and delete food items consumed  and filter based on caloric content.

## Installation

* `git clone <repository-url>`
* `cd meal-tracker`
* `npm install`
* `bower install`
* `gulp serve`
